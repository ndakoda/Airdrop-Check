

 Solana Airdrop Scanner

A lightweight, open-source tool to check Solana wallet eligibility for past and active airdrops. Built with vanilla JavaScript and designed for transparency.

 Features

- Check wallet eligibility for 10+ Solana projects
- Real-time validation of Solana addresses
- Clean, responsive UI
- No data storage – all processing is client-side
- Open source and auditable

 Supported Projects

- Jupiter (JUP)
- Pyth Network (PYTH)
- Wen Token (WEN)
- Bonk (BONK)
- Drift Protocol (DRIFT)
- And more...


 Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Enter a Solana wallet address
4. Click "Check Eligibility"
